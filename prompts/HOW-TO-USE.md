# How To Use This System Design Learning System

This repository is structured for deliberate architectural practice.

## Chat Architecture

Use separate chats for:

1. Roadmap (long-term planning)
2. Weekly Topic Deep Dive
3. Architecture Review
4. Interview Simulation

Do not mix them.

## Philosophy

We prioritize:

- Mental models
- Tradeoff clarity
- Failure modeling
- Sustainable pace
- Long-term growth

Over:

- Flashy diagrams
- Tool hype
- Over-engineering
- Rushed coverage

Consistency compounds.
